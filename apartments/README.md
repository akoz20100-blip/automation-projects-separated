# 🏢 مشروع الشقق — Apartments (ديمورا)

أتمتة استضافة الشقق المفروشة (ديمورا): استقبال الحجوزات، رسائل الضيوف
التلقائية، وصفحة دليل الدخول. يتكوّن من **نظامين**:

```
apartments/
├── messaging-system/   ← نظام الرسائل (الكود الشغّال)
│   └── service/        OCR لقراءة الحجز + إرسال واتساب + إشعارات المالك/العاملة
│                       + توليد صفحة الهبوط + Google Sheets. (Node + TypeScript)
│
└── landing-page/       ← صفحة الهبوط (التصميم + الأصول + التوثيق)
    ├── images/         الصور الفعلية (مدخل، غرف، شعار)
    ├── preview/        معاينات HTML جاهزة
    └── *.md            نظام التصميم + دليل المحتوى + إعداد الفيديو
```

> ملاحظة: صفحة الهبوط يولّدها **نفس** كود نظام الرسائل
> (`messaging-system/service/src/services/landing.ts`) لأنها مرتبطة برسائل
> الواتساب؛ مجلد `landing-page/` يحوي تصميمها وصورها وتوثيقها ومعايناتها.

## ابدأ من هنا
- نظرة عامة على الكود وتشغيله: `messaging-system/README.md`.
- صفحة الدليل (تصميم/صور/تعديل): `landing-page/README.md`.
