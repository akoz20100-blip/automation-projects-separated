## 🔤 الخطوط (Thmanyah / ثمانية)

- **العناوين (H1/H2):** Thmanyah Serif Display — وزن 700، تباعد `-0.02em`.
- **النص وكل الواجهة:** Thmanyah Sans — أوزان 400 / 700.
- **محمّلة محلياً 100%** كـ base64 داخل `../messaging-system/service/src/assets/thmanyahFonts.ts`
  → صفر طلبات خارجية. الملفات + الترخيص في `…/assets/fonts/`.

## 🎨 الألوان (فاتح + أخضر العلامة)

| التوكن | القيمة | الاستخدام |
|---|---|---|
| `--bg` | `#F4F4F6` | خلفية الصفحة (رمادي آبل فاتح) |
| `--card` | `#FFFFFF` | البطاقات |
| `--ink` | `#1D1D1F` | العناوين والنص الأساسي |
| `--body` | `#3F3F43` | نص الفقرات |
| `--muted` | `#86868B` | نص ثانوي |
| `--line` | `rgba(0,0,0,.07)` | حدود رفيعة |
| `--accent` | `#889970` | **الأخضر السَّيج (لون الشعار)** — الأزرار، الأيقونات، النقاط، الرموز، CTA |
| `--accent-dk` | `#5F6B4E` | الأخضر الداكن (مضغوط/رموز) |

- **بلاطات الأيقونات** (`.tile-ic`) بلون العلامة الواحد (تدرّج أخضر ناعم) — لون واحد فقط.
- **لوح الهيرو** (`--art-hero`) توهّج أخضر سَيج ناعم خلف نافذة العرض.
- خلفية الهيرو: صورة داخلية مموّهة خلف ستارة فاتحة (ما تكون بيضاء فقط).

> الألوان قابلة للتغيير: `BRAND_PRIMARY_COLOR` يستبدل الأخضر (أي قيمة غير الافتراضي
> القديم `#0E7C66`). الخلفية تبقى فاتحة.

## 📐 المسافات والزوايا

- زوايا: `--r-sm:14px` / `--r:20px` / `--r-lg:26px` / `--r-xl:32px`.
- حاوية المحتوى: حد أقصى ~680px (موبايل أولاً)، بادنق سخي. ظلال ناعمة فاتحة.

## 🧱 المكوّنات

- **إطار نافذة متصفّح (`.mock`)** بنقاط إشارة المرور — يحيط صورة المبنى (بالحجم الكامل)
  وصورة العرض في الهيرو، فوق لوح أخضر ناعم (`.pad.art-hero`).
- **رؤوس الأقسام (`.sec-head`)**: رقاقة أيقونة خضراء + عنوان Serif عربي + إنجليزي رمادي.
- **مبدّل بيل (`.seg-wrap`)** لاختيار D1/D2 — الفعّال أبيض بظل.
- **بلاطات أيقونات (`.tile-ic`)** بلون العلامة الواحد (للخطوات والخدمات).
- **شريط أرقام كبيرة (`.stats`)**: وقت الدخول · وقت الخروج · دعم 24/7 (حقائق فعلية).
- **قسم التواصل**: زر واتساب + سطر البريد + **بطاقة موقع** بصورة خريطة (مُعاد تلوينها لأخضر)
  وزر «الاتجاهات عبر خرائط جوجل» يفتح الموقع مباشرة.
- بطاقات بيضاء مسطّحة + حدود رفيعة، نقاط قوائم خضراء، أيقونات SVG مضمّنة، زر عائم للموبايل.
- الشعار (DIMORA) يُعرض بفلتر `brightness(0)` ليصير حبري واضح على الخلفية الفاتحة.
- RTL عربي أولاً، إنجليزي كنص ثانوي أصغر، يحترم `prefers-reduced-motion`.

## 🎬 الحركة

- **easing موحّد:** `cubic-bezier(0.16, 1, 0.3, 1)`. ظهور متدرّج للهيرو + كشف عند التمرير.
