# 🏛️ صفحة هبوط ديمورا — Dimora Guest Guide

كل شيء عن صفحة دليل ضيف **ديمورا** في مكان واحد: التصميم، الصور، المراجع، وطريقة التعديل.

> هذا المجلد **مرجع وتوثيق**. الكود الفعلي اللي يولّد الصفحة موجود في خدمة
> العملاء (`service/`) عشان يشتغل ويرتبط برسائل الواتساب — مساراته مذكورة تحت.

---

## 📂 محتويات هذا المجلد

```
dimora-landing/
├── README.md            ← أنت هنا (الفهرس الرئيسي)
├── DESIGN-SYSTEM.md     ← نظام التصميم: ألوان، خطوط، حركة، مراجع عدّة (EDDAH)
├── assets/
│   └── README.md        ← الصور المطلوبة + أسماؤها + وين ترفعها + الفيديو
└── preview/
    ├── dimora-ar.html   ← معاينة جاهزة (افتحها بالمتصفح) — عربي
    └── dimora-en.html   ← معاينة — إنجليزي
```

افتح `preview/dimora-ar.html` بأي متصفح لتشوف التصميم الحقيقي بخطوط Thmanyah (يشتغل بدون نت).

## 🧩 أقسام الصفحة (بالترتيب)

1. **الهيرو** — صورة الواجهة + شعار ديمورا + ترحيب + زر "شاهد فيديو الدخول".
2. **فيديو الدخول** — الأساس اللي يعلّم الضيف الدخول والخروج (في الأعلى مباشرة).
3. **اختيار الشقة** — تبديل D1 / D2 (رقم الوحدة، رمز الدخول، الواي فاي، ملاحظات).
4. **تعليمات الدخول** — صورة المدخل الموضّحة + 4 خطوات + طريقة الدخول.
5. **معلومات الشقة** — واي فاي، وقت الدخول/الخروج، الخدمات والمزايا.
6. **إرشادات الإقامة** — قوانين مختصرة + إرشادات الخروج.
7. **حولك في حي لبن** — معالم، مولات، مطاعم، كافيهات، خدمات يومية.
8. **تحتاج مساعدة؟** — زر واتساب + اتصال الصيانة.

## 🖼️ الصور والفيديو

كل التفاصيل في **`assets/README.md`** — باختصار:
- ارفع الصور في ريبو عام (`akoz20100-blip.github.io/dimora/`) بالأسماء المحددة.
- ارفع الفيديو على YouTube (Unlisted) وحط الرابط.
- بعدها حط الروابط في `guestGuide.ts` (قسم `media`) و `brand.logoUrl`.
